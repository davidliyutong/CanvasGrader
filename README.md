# SpeedGrader

- rarfile
- tqdm

## Usage


```bash
.
├── app.py
└── submission
    ├── 5xxxxxxxxxxxNAME*_*_*.zip
    └── ...
```

Execute

```bash
python app.py
```

```
.
├── app.py
├── submission
│   ├── 5xxxxxxxxxxxNAME_*_*_*_.zip
│   └── ...
├── output
│   ├── 5xxxxxxxxxxxNAME
│   │   ├── src
│   │   ├── include
│   │   └── main
│   └── ...
└── REPORT.md
```
